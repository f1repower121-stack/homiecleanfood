# Menu Images Storage Bucket

The admin menu image upload requires a Supabase Storage bucket named **`menu-images`**.

## Setup

1. Go to **Supabase Dashboard** → **Storage**
2. Click **New bucket**
3. Name: `menu-images`
4. Enable **Public bucket** (so uploaded images are publicly accessible)
5. Save

Alternatively, create via SQL (Storage buckets are created via Dashboard in most setups).
