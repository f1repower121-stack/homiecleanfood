🚀Customer Journey

Week 1: New Customer
├─ Fills intake form
├─ System creates profile + preferences
└─ Receives welcome LINE message

Week 2-3: Meal Planning
├─ Admin reviews preferences
├─ System generates meal plan
├─ Meals match allergies + calories
└─ Ready for first delivery

Week 4+: Regular Deliveries
├─ Daily: Generate meals for day
├─ Send LINE notification morning of
├─ Prepare meals based on preferences
├─ Deliver to address at preferred time
└─ Track delivery status

Month 2+: Flexibility
├─ Pause when traveling
├─ Auto-resume on set date
├─ Buy new packages
└─ Meals combine with existing package
