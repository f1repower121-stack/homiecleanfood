```
Hero Gradient: Premium Green → Deep Navy (left to right)
```

## Animation & Interaction

### 1. Smooth Transitions
```css
All transitions: 0.3s cubic-bezier(0.4, 0, 0.2, 1)
Hover states: Subtle scale, shadow increase
Progress bars: Animate from 0 to final value over 1s
```

### 2. Micro-interactions
```
✓ Button hover: Scale 1.02, shadow grows
✓ Card hover: Lift effect (transform: translateY(-4px))
✓ Progress bar: Smooth fill animation
✓ Modal entrance: Fade + slight scale
✓ Success state: Checkmark animation
```

## Image & Icon Guidelines

### Icons
- Use: Feather or Heroicons (premium, minimal)
- Size: 24px for body, 48px for stats
- Color: Premium Green or Navy
- Style: Consistent stroke width (2px)

### Images
- Meals: Professional photography, bright, appetizing
- Customer photos: Real, authentic, diverse
- Food: Show the quality, presentation, freshness
- Color: Natural, vibrant, warm tones

### Backgrounds
- Solid: Use Cream or Soft Sage
- Gradient: Premium Green → Navy (hero sections)
- Pattern: Minimal, subtle texture (optional)

## Conversion Optimization

### Page Flow
```
1. Hero Section (grab attention)
2. Programs Display (show options)
3. Social Proof (build trust)
4. Customer Testimonials (emotional connection)
5. Guarantee (reduce risk)
6. Call-to-Action (conversion)
```

### Button Hierarchy
```
🟢 Primary (Gold gradient):   Strong action (SELECT, EXPLORE)
🔵 Secondary (Navy outline):  Alternative action (LEARN MORE)
⚫ Tertiary (Text only):       Light action (SKIP, DETAILS)
```

### Form Fields (Conversions)
- Large, clear input fields
- Placeholder text descriptive
- Error states in warm colors (not red)
- Success confirmations
- Progress indicators

## Accessibility

- Contrast ratio: WCAG AA minimum (4.5:1)
- Text size: Minimum 16px for body
- Touch targets: Minimum 44x44px
- Color blindness: Don't rely on color alone
- Mobile: Responsive, tested on all sizes
